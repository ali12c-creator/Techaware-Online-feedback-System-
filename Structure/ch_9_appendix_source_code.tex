This section provides details about the project source code repository.

\textbf{Guidelines for Students:}

\begin{itemize}
    \item Upload your complete project source code to a version control platform 
    such as GitHub, GitLab, or Bitbucket.
    
    \item Ensure the repository includes:
    \begin{itemize}
        \item All source files
        \item Database scripts (if applicable)
        \item Model training and testing code (for ML projects)
        \item Frontend and backend code (for web/mobile projects)
        \item README file with setup instructions
    \end{itemize}
    
    \item The repository must be properly structured with meaningful folder names.
    
    \item Remove unnecessary files (temporary files, cache files, datasets if restricted).
    
    \item If the repository is private, make the repository public before final submission.
    
    \item Paste the final repository link below.
\end{itemize}
